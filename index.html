lqfb-delegation-graph

by Peter Grassberger, GitHub

Description

The lqfb-delegation-graph is an experimental graph view of delegations in LiquidFeedback. Every node in the graph is a member of the selected LiquidFeedback instance. The first number shown in each Node is it's ID, the second number is the number of delegations this member has. Members that do not delegate or are no delegates are not shown. The names can only be shown when you provide an API key, but this is not implemented yet.
